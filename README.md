# Simple-Bank-Client-Management-System-in-C-
This C++ program is a simple client management system.

It allows users to:
Add Clients: Users can input client details like Account Number, Pin Code, Name, Phone, and Account Balance.
View Client List: Displays a list of all clients with their details.
Update and Delete Clients: Users can modify or remove client information.
Perform Transactions: Functions for depositing money to a client's account and checking total balances.
The program runs in the console and uses menus to guide users through these operations. 
It stores client data in a file and allows basic manipulation and management of client records.
